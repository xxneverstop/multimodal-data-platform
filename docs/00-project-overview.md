# 多模态数据基础设施平台项目概述

本项目是面向实验室长期数据采集、管理、质检和复用的数据基础设施平台。当前实验室未来会持续产生来自智能动捕服装、眼镜 IMU/RGB、手套 IMU、足底压力、机器人端等多源数据。如果这些数据长期散落在硬盘、网盘和个人电脑中，会导致采集记录不清、文件版本混乱、数据质量不可追踪、后续算法训练和实验复现困难。因此，本项目第一阶段不追求大而全，而是先建立一套可运行、可扩展的数据底座。

第一阶段以 IMU 数据为主要示例，完成采集任务管理、文件上传、OSS 对象存储、基础质检、质检报告展示这条最小闭环。系统需要支持创建采集任务，登记被试、动作、设备、采集时间等元信息；上传 IMU 原始文件；将文件保存到 OSS；在数据库中记录文件元数据；触发基础质检逻辑；生成结构化质检结果；并在前端或接口中展示报告。

本项目的定位不是普通网盘，也不是单纯的网页管理系统，而是实验室未来多模态动作数据的统一入口。后续可以逐步扩展到多模态时间对齐、数据集版本管理、自动处理流水线、可视化报告、算法训练数据导出等能力。
