两种 OSS 上传链路

后端中转上传

浏览器 → 后端 → OSS

浏览器将文件传给 Spring Boot，再由后端写入 OSS。简单直接，但占用后端带宽和内存。
STS 直传

浏览器 → OSS（直传）

后端通过 STS 签发临时上传凭证，前端直传 OSS 后再回平台登记。高性能，大文件推荐。
MMDP · 交接时需确认当前环境使用哪种方式5 / 19

常见问题排查

AccessDenied
上传被拒绝
检查 Bucket 名称、策略资源路径是否匹配、测试/生产 Bucket 是否混用
STS 获取失败
临时凭证签发失败
检查 Role 是否存在、Trust Policy 是否正确、Role ARN 环境变量
接口报错
页面能打开但 API 不通
Nginx 是否代理 /api、Backend 容器是否运行、.env 变量是否完整
数据库连接失败
MySQL 无法连接
检查 MMDP_DB_URL、容器状态、Compose 网络中主机名一致性、SQL 初始化
AccessKey 创建失败
无权限操作 RAM
检查是否缺少 AliyunSTSAssumeRoleAccess 权限
安全组问题
外部无法访问服务
确认安全组放行 80/443、SSH 来源 IP 限制、容器端口映射正确
